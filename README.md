# OpenFlight

A simple, free iOS flight tracker built with SwiftUI. The app provides a clean interface to look up flight statuses and includes Live Activities support for tracking flights on your Lock Screen or Dynamic Island.

## Features

- Minimalist design focused on a single search bar
- Live Activity integration using ActivityKit
- Placeholder networking layer for flight status data

## Getting Started

1. Open the `OpenFlightApp` folder in Xcode.
2. Add your flight data provider logic in `FlightAPI`.
3. Build and run on iOS 17 or later to test Live Activities.

This codebase serves as a starting point—feel free to expand upon it!
